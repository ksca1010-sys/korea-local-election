# 알선거 페르소나 대응계획 v2

> 원칙: 시뮬레이션 데이터는 존재할 수 없다. 없으면 없다고 보여준다.
> 원칙: 평균 데이터를 실제 데이터인 것처럼 보여주지 않는다.
> 원칙: 알선거의 모든 데이터는 100% 팩트여야 한다.
> docs/ 폴더에 넣고 Claude Code에 "docs/페르소나_대응계획_v2.md를 읽고 P0부터 구현해줘" 지시

---

## 핵심 방침 3가지

```
1. 가짜 데이터는 제거한다. 표기하지 않는다. 제거한다.
   → "시뮬레이션 데이터입니다"라고 붙이는 건 대안이 아니다.
   → 데이터가 없으면 "아직 등록된 데이터가 없습니다"가 정답이다.

2. 전국 평균을 지역 데이터인 것처럼 보여주지 않는다.
   → 영암군 투표율에 전국 평균이 들어가 있으면 그건 거짓이다.
   → 지역별 실제 투표율로 교체한다. 교체할 수 없으면 해당 항목을 비운다.

3. 출처가 없는 데이터는 표시하지 않는다.
   → 모든 숫자에는 출처가 있어야 한다.
   → 선관위, NESDC, 공식 발표 — 이 3가지만 출처로 인정한다.
```

## P0 — 오늘 당장 (데이터 신뢰 복구)

### P0-1. 시뮬레이션 데이터 전면 제거

```
Claude Code 지시:

"코드베이스 전체에서 다음 키워드를 검색해줘:
seededRandom, simulation, isSimulation, 시뮬레이션, 
mock, fake, dummy, sample, 모의, 테스트용

찾은 것을 전부 보고해줘. 그리고:

1. 시뮬레이션으로 생성된 데이터 → 전부 제거 (빈 배열 [] 또는 빈 객체 {})
2. 시뮬레이션 생성 함수/로직 → 전부 제거
3. 데이터가 비면 UI에 '아직 등록된 데이터가 없습니다' 표시
4. 푸터의 '본 사이트의 여론조사 데이터는 시뮬레이션 목적의 모의 데이터입니다' → 제거
5. isSimulation 관련 코드 → 전부 제거 (판별 로직 자체가 필요 없음. 시뮬레이션이 없으니까)

시뮬레이션 데이터를 '표기'하는 것이 아니라 '제거'하는 것이다.
없으면 없다고 보여주는 게 맞다."
```

### P0-2. 역대 투표율 — 전국 평균 → 지역별 실데이터 교체

```
Claude Code 지시:

"data/static/historical_elections.json과 data/mayor_history.json에서
투표율(turnout) 데이터를 전수 확인해줘.

확인 방법:
1. 서로 다른 지역의 같은 회차 투표율이 동일한지 비교
   예: 서울 8회 투표율 == 부산 8회 투표율 == 전남 8회 투표율이면
   이건 전국 평균이 복사된 것

2. 동일한 값이 발견되면 해당 데이터를 전부 제거하고
   선관위 선거통계시스템(info.nec.go.kr) 또는 NEC API에서
   지역별 실제 투표율을 수집해서 교체해줘.

3. 실데이터를 구할 수 없는 회차/지역은
   해당 투표율 필드를 null로 설정하고
   UI에서는 '투표율 데이터 미확보'로 표시해줘.
   절대로 전국 평균을 대입하지 마.

4. 교체 후 커밋 메시지에 출처를 명시해줘:
   'data: 역대 투표율 지역별 실데이터 교체 (출처: 선관위 선거통계시스템)'"
```

### P0-3. 전국 정당지지율 바 — 공표금지 적용

```
Claude Code 지시:

"좌측 사이드바의 '전국 정당지지율' 섹션에도
ElectionCalendar.isPublicationBanned() 체크를 추가해줘.
공표금지 기간이면:

- '공표금지 기간입니다 (5/28~6/3 18:00)' 안내 표시
- 데이터 자체를 렌더링하지 않음 (UI 숨김이 아니라 데이터 미전달)"
```

```

- 탭 전환 시 변경되는 콘텐츠 영역"
```

### P0.5-2. WAI-ARIA 탭 패턴

```
Claude Code 지시:

"정보탭에 WAI-ARIA 탭 패턴을 적용해줘:
- 탭 목록: role='tablist'
- 각 탭: role='tab', aria-selected='true/false', aria-controls='panel-id'
- 각 패널: role='tabpanel', id='panel-id', aria-labelledby='tab-id'
- 키보드: 좌우 화살표로 탭 이동, Enter로 선택"
```

### P0.5-3. 텍스트 대비

```
Claude Code 지시:

"보조 텍스트 색상(--text-muted, --text-tertiary 등)이
WCAG AA 기준 4.5:1 대비를 충족하는지 확인하고
미달이면 밝게 조정해줘.
#5D6B8F 같은 색은 #8B95B0 이상으로 올려줘."
```
