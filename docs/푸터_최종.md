## 3. Claude Code 지시문

```
"docs/푸터_최종.md를 읽고 구현해줘.

1. index.html 하단의 기존 푸터를 삭제하고
   1절의 HTML을 </main> 앞 또는 </body> 앞에 삽입해줘.

2. css/style.css 맨 하단에 2절의 CSS를 추가해줘.

3. index.html <head>에 아래 폰트 링크가 없으면 추가:
   Material Symbols Outlined
   Inter + Noto Sans KR

4. 기존 푸터의 '6.3 전국지방선거' 관련 텍스트 전부 제거.
   '시뮬레이션 목적의 모의 데이터입니다' 문구도 제거.

5. GitHub 저장소 URL은 실제 저장소 주소로 교체.

6. App.showScheduleModal과 App.showAboutModal이
   아직 없으면 빈 함수로 만들어둬:
   App.showScheduleModal = function() {};
   App.showAboutModal = function() {};

7. 기존 style.css에 충돌하는 footer 관련 스타일이 있으면
   .al-footer 접두사를 사용해서 격리해줘."
```

## 4. DESIGN.md 원칙 체크리스트

```
✅ No-Line Rule: footer-main ↔ footer-bottom 사이 border 없음
   → #101415 → #191c1e 배경색 전환으로 구분
✅ Ghost Borders: footer-sources의 border-top만
   → rgba(65, 71, 84, 0.15) = outline-variant 15%
✅ 순수 검정 금지: #101415, #191c1e 사용
✅ 100% 흰색 금지: #e0e3e5 (제목), #c1c6d6 (링크), #8b919f (보조)
✅ 전환 200ms ease-in-out: 모든 a:hover
✅ 폰트: Inter + Noto Sans KR
✅ letter-spacing: 라벨 0.06em, 메타 0.02em
✅ 뉴스레터 → 데이터 출처로 교체 (백엔드 없음)
✅ 언어 선택기 제거 (한국어만)
✅ 소셜: GitHub SVG 1개만
✅ 비영리 고지: "특정 정당이나 후보를 지지·추천하지 않습니다"
```
