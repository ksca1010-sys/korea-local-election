# Plan 01: NEC 본후보 API 수집 실행

## Objective
본후보 등록 마감(5/14) 직후 NEC API로부터 광역단체장·교육감·기초단체장 후보 데이터를 수집하고, 기존 예비후보 데이터와 병합하여 `data/candidates/` JSON을 업데이트한다.

## Context
Phase 6에서 구현된 `fetch_nec_candidates.py` 파이프라인을 5/14 당일 실행하는 절차다. 날짜 게이팅(5/14 이전 API 미호출)이 해제되는 첫 날이므로, dry-run으로 API 응답 구조를 먼저 확인한 후 본 실행한다. GitHub Actions `update-candidates.yml`이 매일 09:00 KST에 자동 실행되지만, 수동 실행으로 즉시 결과를 확인하고 ballotNumber 필드명 문제를 조기에 발견·대응하는 것이 목적이다.

## Execution Date
**2026-05-14** -- 본후보 등록 마감일 직후 실행

## Prerequisites
- [ ] Phase 6 코드가 main 브랜치에 병합되어 있다
- [ ] `scripts/candidate_pipeline/fetch_nec_candidates.py` 파일이 존재한다
- [ ] `scripts/candidate_pipeline/nec_precand_sync.py` 파일이 존재한다
- [ ] `data/candidates/governor.json`, `superintendent.json`, `mayor_candidates.json`이 존재한다

- [ ] requests 패키지 설치 확인: `python3 -c "import requests; print(requests.__version__)"`

### 5. 즉시 결과 확인

- [ ] **[검증]** 후보 수 카운트
  ```
  python3 -c "
  import json
  for f in ['governor', 'superintendent', 'mayor_candidates']:
      with open(f'data/candidates/{f}.json') as fh:
          data = json.load(fh)
      total = sum(len(v) for v in data['candidates'].values()) if isinstance(list(data['candidates'].values())[0], list) else sum(len(c) for region in data['candidates'].values() for c in (region.values() if isinstance(region, dict) else [region]))
      print(f'{f}: {total}명')
  "
  ```

- [ ] **[검증]** NOMINATED 상태 후보 수 확인
  ```
  python3 -c "
  import json
  for f in ['governor', 'superintendent']:
      with open(f'data/candidates/{f}.json') as fh:
          data = json.load(fh)
      nominated = sum(1 for region in data['candidates'].values() for c in region if c.get('status') == 'NOMINATED')
      total = sum(len(region) for region in data['candidates'].values())
      print(f'{f}: NOMINATED {nominated}/{total}')
  "
  ```
  - 확인: NOMINATED 수가 기존 DECLARED 수와 유사하거나 약간 적어야 정상

- [ ] **[검증]** ballotNumber 채워진 후보 수
  ```
  python3 -c "
  import json
  for f in ['governor', 'superintendent']:
      with open(f'data/candidates/{f}.json') as fh:
          data = json.load(fh)
      with_ballot = sum(1 for region in data['candidates'].values() for c in region if c.get('ballotNumber') and c['ballotNumber'] > 0)
      nominated = sum(1 for region in data['candidates'].values() for c in region if c.get('status') == 'NOMINATED')
      print(f'{f}: ballotNumber 있음 {with_ballot}/{nominated} NOMINATED')
  "
  ```
  - 확인: NOMINATED 후보 전원에게 ballotNumber가 있어야 정상
  - **실패(0명):** ballotNumber 필드명 불일치 -- Plan 02 "ballotNumber 필드명 패치" 절차로 이동

- [ ] **[검증]** unmatched 후보 확인
  ```
  python3 -c "
  import json
  with open('data/candidates/unmatched_candidates.json') as f:
      data = json.load(f)
  count = len(data.get('candidates', []))
  print(f'unmatched: {count}명')
  if count > 0:
      for c in data['candidates'][:5]:
          print(f'  - {c.get(\"reason\", \"unknown\")}: {c.get(\"candidate\", {}).get(\"name\", \"?\")}')
  "
  ```
  - 확인: `unmatched: 0명`이면 완전 자동 처리 성공
  - **0보다 크면: Plan 02 "unmatched 후보 수동 처리" 절차로 이동**
