# Plan 03-03: 탭 스켈레톤 스크린 + 모바일 패널 스와이프 닫기

## Goal

데이터 로딩 중 탭 영역에 스켈레톤 shimmer 효과를 표시하고, 모바일에서 사이드 패널을 아래로 스와이프하여 닫는 터치 인터랙션을 구현한다.

## must_haves

- 탭 콘텐츠 영역에 `.skeleton-shimmer` 클래스의 플레이스홀더가 데이터 로딩 중 표시됨
- CSS에 shimmer 애니메이션 키프레임이 정의됨
- 모바일 패널 상단에 `.drag-handle` 요소가 존재함
- touchstart/touchmove/touchend 이벤트로 스와이프 거리 계산 후 closePanel() 호출

   ```js
   function showSkeleton(tabName) {
     const container = document.getElementById('tab-content');
     if (!container) return;
     let html = '<div class="skeleton-container">';
     html += '<div class="skeleton-line title"></div>';
     if (tabName === 'polls' || tabName === 'history') {
       html += '<div class="skeleton-chart"></div>';
     }
     for (let i = 0; i < 5; i++) {
       const w = ['short', 'medium', 'long'][i % 3];
       html += `<div class="skeleton-line ${w}"></div>`;
     }
     html += '</div>';
     container.innerHTML = html;
   }
   ```

3. `js/app.js`의 `init()` 함수 내부 (또는 패널이 열릴 때)에 터치 이벤트 핸들러 등록:
   ```js
   function _initSwipeClose() {
     const panel = document.getElementById('info-panel')
       || document.querySelector('.info-panel');
     if (!panel) return;
     let startY = 0, currentY = 0, isDragging = false;
     const THRESHOLD = 100; // px

     panel.addEventListener('touchstart', (e) => {
       // 패널 상단 80px 영역에서만 스와이프 시작
       const rect = panel.getBoundingClientRect();
       const touchY = e.touches[0].clientY - rect.top;
       if (touchY > 80) return;
       startY = e.touches[0].clientY;
       isDragging = true;
       panel.style.transition = 'none';
     }, { passive: true });

     panel.addEventListener('touchmove', (e) => {
       if (!isDragging) return;
       currentY = e.touches[0].clientY;
       const diff = currentY - startY;
       if (diff > 0) { // 아래로만
         panel.style.transform = `translateY(${diff}px)`;
       }
     }, { passive: true });

     panel.addEventListener('touchend', () => {
       if (!isDragging) return;
       isDragging = false;
       const diff = currentY - startY;
       panel.style.transition = 'transform 0.3s ease';
       if (diff > THRESHOLD) {
         panel.style.transform = `translateY(100%)`;
         setTimeout(() => {
           closePanel();
           panel.style.transform = '';
           panel.style.transition = '';
         }, 300);
       } else {
         panel.style.transform = '';
       }
       startY = 0; currentY = 0;
     });
   }
   ```

4. `init()` 함수 끝부분에서 `_initSwipeClose()` 호출

5. 모바일 패널의 CSS에 `will-change: transform` 추가하여 GPU 가속:
   ```css
   @media (max-width: 768px) {
     .info-panel, #info-panel { will-change: transform; }
   }
   ```
</action>
