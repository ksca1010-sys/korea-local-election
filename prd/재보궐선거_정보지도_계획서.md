# 2026 재보궐선거 정보지도 구현 계획서

> 작성일: 2026.03.09 | 선거일: 2026.06.03 (제9회 전국동시지방선거 동시 실시)

---

## 1. 프로젝트 개요

6·3 지방선거와 동시에 치러지는 **국회의원 재보궐선거** 지역을 한눈에 파악할 수 있는 **인터랙티브 전국 지도**를 구현한다. 광역시도 지도에서 해당 광역을 선택하면, 상세 지도에서 **재보궐선거가 확정된 선거구만 활성화**되는 2단계 드릴다운 방식.

## 3. UI/UX 설계

### 3.1 2단계 드릴다운 구조

```
[1단계] 전국 광역시도 지도
  → 재보궐선거가 있는 광역시도만 컬러 강조 (인천, 경기, 충남, 전북)
  → 나머지 광역시도는 비활성(회색조)

[2단계] 광역시도 상세 지도
  → 해당 광역시도 내 선거구 표시
  → 재보궐선거 확정 선거구만 활성화 (컬러 + 클릭 가능)
  → 비대상 선거구는 비활성(회색조)
  → 활성 선거구 클릭 시 사이드패널에 상세 정보 표시
```

### 3.2 활성 선거구 클릭 시 표시 정보

- 선거구명, 사유 (보궐/재선거), 전임 의원 및 소속 정당
- 주요 후보군 (확정 시)
- 선거구 인구/유권자 수
- 이전 선거 결과 요약

## 4. 기술 스택

| 구분 | 기술 | 비고 |
|------|------|------|
| 프레임워크 | **React (JSX 아티팩트)** | 바이브코딩 빠른 프로토타이핑 |
| 지도 렌더링 | **D3.js + TopoJSON** | 광역시도/선거구 경계 SVG 렌더링 |
| 지도 데이터 | 행정구역 TopoJSON | 통계청 행정경계 + 선관위 선거구 데이터 |
| 스타일링 | Tailwind CSS | 반응형, 다크/라이트 |
| 상태관리 | React useState/useReducer | 선택된 광역, 선거구 상태 |

## 6. 구현 단계

### Phase 1: MVP (1~2일) ← 현재 단계
- [ ] React 아티팩트로 전국 광역시도 지도 렌더링 (SVG 하드코딩)
- [ ] 재보궐 대상 광역시도 활성화/비활성화
- [ ] 광역 클릭 → 상세 지도 전환 (대상 선거구만 하이라이트)
- [ ] 선거구 클릭 → 사이드패널 정보 표시
- [ ] 하단 확정 지역 카드 리스트

### Phase 2: 데이터 보강 (선거일까지 지속)
- [ ] 후보 확정 시 후보 정보 업데이트
- [ ] 추가 확정 선거구 반영 (4월 30일 마감)
- [ ] 여론조사 데이터 연동 (선택)
- [ ] 모바일 반응형 최적화

### Phase 3: 6·3 선거 통합 (선택)
- [ ] 기존 6·3 지방선거 정보지도와 통합
- [ ] 재보궐 + 지선 동시 표시 토글

## 7. 지도 데이터 전략

실제 TopoJSON 파일 없이도 **SVG 기반 간이 지도**로 MVP 제작 가능:

1. **전국 광역시도**: 17개 광역시도를 SVG path로 간략화하여 하드코딩
2. **광역시도 내 선거구**: 대상 선거구만 개략적 경계를 SVG로 표현
3. 추후 정확한 선거구 경계는 선관위/통계청 데이터로 교체
