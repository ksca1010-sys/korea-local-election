# 재보궐선거 뉴스 알고리즘 로직

> 참조: 기존 뉴스탭 알고리즘 설계 계획서 (2026.03.09)
> 목적: 기존 6개 선거종류(시도지사~기초의원) 로직에 7번째 선거종류 "국회의원 재보궐"을 추가

---

## 1. 재보궐이 기존 로직과 다른 점

| 항목 | 광역/기초 선거 | 재보궐선거 |
|------|-------------|-----------|
| **지역 단위** | 광역시도 / 기초지자체 | 국회의원 선거구 (읍면동 조합) |
| **관심도** | 시도지사 > 기초단체장 > 의원 | **전국뉴스급** (국회의원이므로) |
| **뉴스 소스** | 기초=지역신문, 광역=전국지 | **전국지+지역지 혼합** |
| **사퇴 사유** | 해당 없음 | 뉴스 자체가 됨 (왜 재보궐인지) |
| **전국 연계** | 약함 | 강함 (정당 대결 구도, 정국 영향) |

핵심: 재보궐은 국회의원급이라 **전국지가 Tier 1이면서 동시에 해당 선거구 지역신문도 Tier 1**인 혼합 구조

## 3. 점수 산출 공식 — 재보궐 전용 가중치

기존 공식:
```
Score = 0.40×Time + 0.30×Relevance + 0.20×Credibility + 0.10×Engagement
```

재보궐 전용 조정:
```
Score = 0.35×Time + 0.35×Relevance + 0.20×Credibility + 0.10×Engagement
```

변경 이유: 재보궐은 5개 지역만 있어서 "정확한 선거구 매칭"이 더 중요함.
지방선거는 수백 개 지역이라 최신성이 중요하지만,
재보궐은 5개뿐이니까 관련성(Relevance)을 Time과 동일하게 올림.

## 4. RelevanceScore — 재보궐 전용

```javascript
function calcByelectionRelevance(article, district) {
  let score = 0;
  
  // 1) 선거구 정확 매칭 (0~0.5)
  //    "인천 연수갑"이 제목이나 본문에 있으면 최고점
  if (matchesExactDistrict(article, district.district))     score += 0.5;
  //    "인천 연수구"만 있으면 차선
  else if (matchesMunicipality(article, district.municipality)) score += 0.35;
  //    "인천" 광역시도만 있으면
  else if (matchesMetro(article, district.metro))           score += 0.15;
  //    "재보궐" 일반 뉴스
  else if (article.includes("재보궐"))                       score += 0.1;
  
  // 2) 선거종류 매칭 (0~0.3)
  //    "국회의원 재보궐"이면 최고
  if (matchesKeywords(article, ["국회의원 재보궐", "재보궐선거"])) score += 0.3;
  //    "재보궐"만 있으면
  else if (matchesKeywords(article, ["재보궐"]))              score += 0.2;
  //    "6.3 지방선거"에 재보궐이 포함된 기사
  else if (matchesKeywords(article, ["6.3 선거", "지방선거"])) score += 0.05;
  
  // 3) 사퇴 사유 관련 뉴스 보너스 (0~0.2)
  //    재보궐은 "왜 이 선거가 열리는가"가 중요한 맥락
  if (matchesKeywords(article, getReasonKeywords(district)))  score += 0.15;
  
  return Math.min(score, 1.0);
}

// 사퇴 사유별 키워드
function getReasonKeywords(district) {
  const reasonMap = {
    "인천 연수구 갑": ["박찬대", "인천시장 출마", "사퇴"],
    "인천 계양구 을": ["이재명", "대통령 당선", "궐위"],
    "경기 평택시 을": ["이병진", "선거법 위반", "유죄"],
    "충남 아산시 을": ["강훈식", "대통령비서실장", "사퇴"],
    "전북 군산·김제·부안 갑": ["신영대", "경선조작", "유죄"]
  };
  return reasonMap[district.name] || [];
}
```

## 5. CredibilityScore — 재보궐 전용 Tier

재보궐은 국회의원급이라 전국지와 지역지 모두 Tier 1.

```javascript
const BYELECTION_CREDIBILITY = {
  // 전국지 (국회의원급이므로 Tier 1)
  tier1_national: {
    score: 1.0,
    media: [
      // 5대 종합일간지
      "조선일보", "중앙일보", "동아일보", "한겨레", "경향신문",
      // 방송 3사
      "KBS", "MBC", "SBS",
      // 통신사
      "연합뉴스", "뉴스1"
    ]
  },
  
  // 해당 선거구 지역 일간지 (재보궐에서도 Tier 1)
  tier1_regional: {
    score: 1.0,
    // 각 선거구의 regionalMedia 필드 참조
    // 예: 인천 연수갑 → 인천일보, 기호일보, 인천투데이
  },
  
  // 정치전문 인터넷매체
  tier2: {
    score: 0.8,
    media: [
      "오마이뉴스", "프레시안", "미디어오늘", "뉴스타파",
      "시사IN", "한국일보", "서울신문",
      "데일리안", "뉴데일리", "미디어펜"
    ]
  },
  
  // 기타 인터넷 매체
  tier3: {
    score: 0.5,
    media: [] // 위에 없는 나머지
  }
};

function getByelectionCredibility(sourceName, district) {
  // 해당 선거구 지역매체이면 Tier 1
  if (district.regionalMedia.includes(sourceName)) return 1.0;
  // 전국지이면 Tier 1
  if (BYELECTION_CREDIBILITY.tier1_national.media.includes(sourceName)) return 1.0;
  // 정치전문매체이면 Tier 2
  if (BYELECTION_CREDIBILITY.tier2.media.includes(sourceName)) return 0.8;
  // 나머지 Tier 3
  return 0.5;
}
```

## 7. 재보궐 뉴스 특수 로직

### 7.1 "왜 재보궐인가" 맥락 뉴스 상단 고정

재보궐 선거구를 처음 클릭한 유저에게 가장 먼저 보여줘야 할 정보:
"이 선거가 왜 열리는지"

```javascript
function getByelectionContextNews(district) {
  // 사퇴/궐위 사유를 설명하는 기사를 별도 섹션으로 표시
  // "이 선거가 열리는 이유" 카드
  return {
    type: "context",
    title: `이 선거가 열리는 이유`,
    description: district.reason,
    // 관련 뉴스 1~2건을 고정 표시
    pinnedArticles: searchByKeywords(getReasonKeywords(district), limit=2)
  };
}
```

### 7.2 5개 선거구 교차 뉴스

재보궐 5개 지역을 묶어서 다루는 기사가 많음.
"6.3 재보궐 5곳 판세 분석" 같은 기사는 모든 선거구에 노출.

```javascript
function isCrossByelectionNews(article) {
  // 2개 이상 선거구가 언급되거나 "재보궐 전체" 뉴스
  const mentionedDistricts = BYELECTION_DISTRICTS.filter(d => 
    article.matches(d.searchKeywords.primary)
  );
  return mentionedDistricts.length >= 2 || 
         article.matches(["재보궐 5곳", "재보궐 전체", "재보궐 판세"]);
}

// 교차 뉴스는 모든 재보궐 선거구에 노출하되 
// 해당 선거구 전용 뉴스보다 아래에 배치
```

### 7.3 전국 정치 뉴스 연계

재보궐은 정당 대결 구도가 강해서 전국 정치 뉴스와 연결됨.
"여야 재보궐 총력전" 같은 기사는 재보궐 뉴스에 포함.

```javascript
function isNationalByelectionNews(article) {
  return article.matches(["재보궐"]) && 
         article.matches(["여당", "야당", "민주당", "국민의힘", "정국", "국회"]);
}

// 전국 연계 뉴스는 관련성 점수를 0.1로 낮추되 노출은 시킴
// → 해당 선거구 전용 뉴스가 항상 먼저 나오고, 그 아래에 전국 뉴스
```
