## 3. 실행 순서 (위험도 관리)

```
Phase 1 (LOW)     → 정적 설정 3개 파일
Phase 2 (MEDIUM)  → js/data.js 1개 파일
Phase 3 (MEDIUM)  → 후보 데이터 6개 파일
Phase 4 (LOW)     → 미디어 2개 파일
  ── 여기서 중간 검증: 대구 클릭 시 군위군 목록 표시 확인 ──
Phase 5 (HIGH)    → 역대선거 5개 파일
  ── 여기서 검증: 역대비교 탭 정상 로드 확인 ──
Phase 6 (HIGHEST) → 지도 파일 10개
  ── 최종 검증: 지도에서 대구 선택 시 군위군 폴리곤 렌더링 ──
Phase 7 (LOW)     → 스크립트 정리
```
