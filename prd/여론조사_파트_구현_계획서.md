# 여론조사 파트 구현 계획서

> 작성일: 2026.03.09 | 핵심 목표: "돌출 조사 1건이 아닌, 추이로 읽는 여론"

---

## 1. 준호의 방향성 검토

### 1.1 문제 인식 — 정확하다

> "돌출적인 여론조사 1건이 뉴스 헤드라인에 실려 선거에 영향을 미치는 현상이 공정한 선거를 해친다"

이 문제의식은 학술적으로도 검증된 현상이다.

**승자편승 효과(Bandwagon Effect)**: 여론조사에서 우세한 후보에게 추가 지지가 몰리는 현상으로, 단일 조사 결과가 과도하게 보도될 때 더 심화된다. (김현정, 언론정보연구 2018)

**조사기관 편향(House Effect)**: 박종희 서울대 교수의 연구(조사연구, 2013)에서 18대 대선 여론조사 22개 기관 중 2개 기관이 특정 후보에게 평균 2.5~4.3%의 체계적 편향을 보인 것으로 확인되었다. 같은 방법론을 20대 대선에도 적용한 후속 연구(이신애·박찬경, 한국언론학보 2024)에서도 기관 편향과 보도 빈도 사이의 유의미한 관계가 확인되었다.

**MBC 여론M의 접근**: 정확히 준호와 같은 문제의식에서 출발한 서비스다. 베이지안 상태공간 모형으로 조사기관 편향을 보정하고 여론의 추이(trend)를 추정한다. 핵심 메시지도 동일하다 — "들쭉날쭉한 여론조사, 어떤 게 맞을까요?"

### 1.2 방향 평가 — 효과적이다, 단 보완점 있음

**강점**:
- 동일 기관 추이를 보여주는 것은 하우스 이펙트를 유권자가 직관적으로 파악하게 하는 가장 쉬운 방법
- 여심위 등록 조사만 사용하는 것은 데이터 신뢰도 확보의 정석
- 60개 기관 정보를 알고리즘에 저장하는 것은 기관별 메타데이터 활용의 기반

**보완 필요점**:
1. 동일 기관 추이만으로는 **기관 간 비교**가 안 된다 → 기관별 편향을 보정한 **통합 추세선**이 필요
2. 조사방법(ARS vs 전화면접)에 따른 체계적 차이를 유저에게 명시해야 한다
3. 표본오차 시각화가 빠지면 "접전"과 "격차"를 오판하게 된다

## 2. 레퍼런스 분석

### 2.1 MBC 여론M (poll-mbc.co.kr)

```
방법: 베이지안 상태공간 모형 (박종희 서울대 교수 공동 연구)
핵심: 조사기관별 편향(house effect)을 추정·보정하여 "진짜 추세"를 추출
장점: 학술적 신뢰도 최고, 신뢰구간(confidence interval) 시각화
한계: 전국 단위·대선/총선 위주, 지방선거 226개 기초지역까지 커버 불가
→ 준호의 서비스는 지방선거 지역구 단위까지 내려간다는 점에서 차별화
```

### 2.2 미국 FiveThirtyEight (현 538.com by ABC News)

```
방법: 조사기관별 등급(A+ ~ F), 과거 정확도 기반 가중치 부여
핵심: 모든 여론조사를 등급별 가중평균하여 단일 지지율 추정
장점: 유권자가 가장 이해하기 쉬운 형태 (하나의 숫자)
한계: 한국은 미국처럼 기관 수가 많지 않아 가중평균의 통계적 안정성이 낮음
→ 기관 등급 개념은 차용 가능하되, 한국 실정에 맞게 조정 필요
```

### 2.3 RealClearPolitics (RCP Average)

```
방법: 최근 N개 조사의 단순 평균 (RCP Average)
핵심: 복잡한 모형 없이 "여러 조사의 평균"만 제시
장점: 단순 명료, 유저가 직관적으로 이해
한계: 기관 편향 무보정, 오래된 조사와 최신 조사를 동일 비중
→ MVP 단계에서 가장 빠르게 구현할 수 있는 방법
```

### 3.2 기관 신뢰도 등급 산출

```javascript
function calcReliabilityGrade(institution) {
  const e = institution.trackRecord.avgError;
  const n = institution.trackRecord.elections.length;
  
  // 과거 실적이 3회 미만이면 등급 미부여
  if (n < 3) return "N/R";  // Not Rated
  
  if (e <= 2.0) return "A";   // 평균 오차 2% 이내
  if (e <= 3.5) return "B";   // 평균 오차 3.5% 이내
  if (e <= 5.0) return "C";   // 평균 오차 5% 이내
  return "D";                  // 5% 초과
}

// 조사방법 보정: 전화면접은 ARS보다 과거 정확도가 높은 경향
// (김영원 2014 연구 기반)
function getMethodBonus(method) {
  if (method === "전화면접") return +0.1;  // 신뢰도 소폭 가산
  if (method === "ARS") return 0;
  return 0;
}
```

## 5. 핵심 알고리즘 — 추이 분석

### 5.1 레이어 1: 동일 기관 추이선 (Same-Pollster Trend)

준호의 원래 구상. 가장 기본이면서 가장 중요한 시각화.

```
원리: 같은 기관이 같은 지역에서 반복 조사한 결과를 시간순으로 연결
효과: 조사방법·표본구성이 동일하므로 "순수한 여론 변화"에 가장 가까움
```

```javascript
function getSamePollsterTrend(polls, institutionId, region) {
  return polls
    .filter(p => p.institutionId === institutionId && p.region === region)
    .sort((a, b) => new Date(a.surveyDate.end) - new Date(b.surveyDate.end))
    .map(p => ({
      date: p.surveyDate.end,
      results: p.results,
      gap: p.gap,
      marginOfError: p.method.marginOfError
    }));
}
```

### 5.2 레이어 2: 조사방법별 분리 표시 (Method Split)

ARS와 전화면접은 체계적 차이를 보이므로 분리 표시한다.

```
같은 지역의 여론조사를 ARS / 전화면접으로 분리
→ 각 그룹 내에서 추이선을 그림
→ 유저가 "ARS에서는 접전인데 전화면접에서는 격차가 크다" 등을 직관적으로 파악
```

```javascript
function groupByMethod(polls, region) {
  const phoneInterview = polls.filter(p => 
    p.region === region && p.method.type === "전화면접"
  );
  const ars = polls.filter(p => 
    p.region === region && p.method.type === "ARS"
  );
  return { phoneInterview, ars };
}
```

### 5.3 레이어 3: 통합 추세선 (Consensus Trend)

모든 기관의 조사를 통합하여 "지금 여론은 대략 이 정도"라는 추정치를 제공한다.

**MVP 방식 — 가중 이동평균**:

```javascript
function calcConsensusTrend(polls, region, windowDays = 14) {
  // 최근 N일 이내 조사만 포함
  const cutoff = new Date(Date.now() - windowDays * 86400000);
  const recentPolls = polls.filter(p => 
    p.region === region && new Date(p.surveyDate.end) >= cutoff
  );
  
  if (recentPolls.length < 2) return null;  // 조사 2건 미만이면 추세 미제공
  
  // 가중치 = (1) 최신성 × (2) 기관 신뢰도 × (3) 표본크기
  const weighted = recentPolls.map(p => {
    const inst = getInstitution(p.institutionId);
    const recency = calcRecencyWeight(p.surveyDate.end);       // 최신일수록 높음
    const reliability = calcReliabilityWeight(inst.trackRecord); // A=1.0, B=0.8, C=0.6
    const sampleWeight = Math.sqrt(p.method.sampleSize / 1000); // 표본크기 보정
    
    return {
      ...p,
      weight: recency * reliability * sampleWeight
    };
  });
  
  // 후보별 가중평균
  const candidates = getCandidateList(weighted);
  const consensus = {};
  
  for (const candidate of candidates) {
    const totalWeight = weighted.reduce((sum, p) => sum + p.weight, 0);
    consensus[candidate] = weighted.reduce((sum, p) => {
      const result = p.results.find(r => r.candidateName === candidate);
      return sum + (result ? result.support * p.weight : 0);
    }, 0) / totalWeight;
  }
  
  return {
    estimates: consensus,
    pollCount: recentPolls.length,
    windowDays,
    disclaimer: "최근 14일간 조사의 가중평균이며, 개별 조사의 정확한 수치와 다를 수 있습니다."
  };
}
```

**고도화 방식 — 베이지안 상태공간 모형** (Phase 3):

```
MBC 여론M과 동일한 방법론 (박종희 교수 모형)
→ 각 조사기관의 house effect를 잠재변수로 추정
→ house effect를 제거한 "진짜 여론 추이"를 추출
→ 구현 복잡도 높음, R/Stan 필요 → 선거 전 고도화 목표
```

### 5.4 돌출 조사 감지 (Outlier Detection)

준호의 핵심 문제의식을 직접 구현하는 기능.

```javascript
function detectOutlier(newPoll, region) {
  // 최근 7일 동일 지역 조사들의 평균과 표준편차
  const recentPolls = getRecentPolls(region, 7);
  if (recentPolls.length < 3) return { isOutlier: false };
  
  const avgGap = mean(recentPolls.map(p => p.gap));
  const sdGap = standardDeviation(recentPolls.map(p => p.gap));
  
  // 새 조사의 격차가 평균에서 2σ 이상 벗어나면 돌출
  const zScore = Math.abs(newPoll.gap - avgGap) / sdGap;
  
  if (zScore >= 2.0) {
    return {
      isOutlier: true,
      severity: zScore >= 3.0 ? "extreme" : "moderate",
      label: `이 조사의 격차(${newPoll.gap}%p)는 최근 평균(${avgGap.toFixed(1)}%p)에서 크게 벗어납니다.`,
      suggestion: "동일 기관의 추이를 확인하거나, 다른 기관의 조사와 비교해 보세요."
    };
  }
  
  return { isOutlier: false };
}
```

### 6.1 기관별 편향 차트

## 8. 선거종류 × 지역별 여론조사 커버리지

| 선거종류 | 여론조사 빈도 | 대응 전략 |
|----------|-------------|-----------|
| **시도지사** | 풍부 (주 2~5건) | 추이차트 + 통합추세선 + 기관비교 모두 가능 |
| **교육감** | 희소 (월 1~2건) | 조사 목록 위주, 추이차트는 데이터 축적 후 |
| **기초단체장** | 매우 희소 (선거구별 0~3건) | 있으면 표시, 없으면 "조사 미실시" 안내 |
| **광역/기초의원** | 거의 없음 | 여론조사 탭 비활성화 → 뉴스탭으로 대체 |
| **재보궐선거** | 중간 (국회의원급이므로) | 추이차트 가능, 기관비교 가능 |

```
여론조사 데이터가 0건인 지역:
→ "이 지역에 등록된 여론조사가 아직 없습니다."
→ 상위 광역시도 시도지사 여론조사로 안내
```

## 9. 법적 준수사항

```
1. 여심위 등록 조사만 사용 (공직선거법 제108조)
   → 미등록 조사 결과 공표 시 과태료 1,500만원

2. 공표 시 필수 기재사항 (선거여론조사기준 제13조)
   → 조사의뢰자, 조사기관, 조사지역, 조사일시
   → 조사대상, 조사방법, 표본크기, 피조사자선정방법
   → 응답률, 가중값, 표본오차, 질문내용
   → 우리 서비스는 이 정보를 모두 "조사 원문 보기"로 연결

3. 공표금지 기간 (선거일 전 6일 ~ 투표마감)
   → 2026.05.28 00:00 ~ 2026.06.03 투표마감
   → 이 기간에는 여론조사 탭 전체를 비활성화
   → "선거법에 따라 여론조사 결과를 표시할 수 없습니다" 안내

4. 왜곡 금지 (공직선거법 제108조의2)
   → 결과를 편집·가공하여 특정 후보에 유리하게 왜곡 금지
   → 우리의 통합추세선은 "분석 결과"이지 "조사 결과"가 아님을 명시
```
