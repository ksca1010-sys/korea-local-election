## Out of Scope

| Feature | Reason |
|---------|--------|
| 사용자 계정/로그인 | 정보 조회 전용, 인증 불필요 |
| React/Vue 등 프레임워크 전환 | 바닐라 JS 유지 원칙, 전환 비용 대비 효과 없음 |
| 후보자 직접 등록 | 선관위 공식 데이터만 허용 (헌법 제1조) |
| 선거 예측/AI 수치 | 허위 데이터 방지 원칙 (헌법 제2조) |
| 서버사이드 렌더링 | Cloudflare Pages 정적 호스팅 제약 |
